# Panel de Administración

Solo disponible para usuarios con permisos de administrador.

---

## Funciones del Panel Izquierdo

| Función | Descripción |
|---|---|
| **Seleccionar PDF** | Elige un archivo del repositorio local. |
| **Indexar Conocimiento** | Procesa documentos y genera embeddings. |
| **Exportar Índice** | Guarda el índice como respaldo. |
| **Importar Índice** | Carga un índice previamente guardado. |
| **Limpiar Colección** | Borra todo el conocimiento del sistema. |
| **Ocultar Panel** | Vuelve al modo usuario común. |

---

### Flujo recomendado al agregar nueva información

1. Seleccionar PDF.
2. Indexar Conocimiento.
3. (Opcional) Exportar Índice como respaldo.

---

⚠ **ADVERTENCIA**
Si se usa *Limpiar Colección*, será necesario **reindexar todo**.
