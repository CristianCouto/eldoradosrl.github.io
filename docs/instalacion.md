# Instalación del Sistema

El sistema funciona en una PC local con LM Studio + Backend Flask + Frontend React.

---

## Requisitos recomendados

| Componente | Especificación sugerida |
|---|---|
| RAM | 32 GB |
| CPU | Ryzen 7 5700G o similar |
| GPU | Placa Gráfica dedicada |
| OS | Windows 10 / 11 |

---

## Pasos de Inicio

1. **Abrir LM Studio**
   - Seleccionar modelo configurado.
   - Iniciar servidor local (`Enable API Server`).

2. **Iniciar Backend**

**python app.py**

3. **Iniciar Frontend**
   
**npm start**

5. Acceder en navegador:
http://localhost:3000
