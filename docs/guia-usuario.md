# Guía de Usuario  
**Portal de Soporte Interno - El Dorado SRL**

Este sistema es un **asistente virtual corporativo**, diseñado para responder consultas basadas en documentos internos de El Dorado.  
Permite agilizar soporte técnico, procedimientos internos, manuales de máquinas y gestión de conocimiento.

---

## 1) Acceso

El sistema ofrece dos modalidades:

| Modo | Descripción |
|---|---|
| **Chatbot** | Uso normal. Permite hacer consultas y recibir respuestas. |
| **Administrador** | Permite cargar nuevos documentos y administrar el conocimiento. |

---

## 2) Interfaz Principal

- **Cuadro de texto** → Escriba su consulta.
- **Botón de micrófono** → Permite dictar por voz (presione para iniciar, presione nuevamente para finalizar).
- **Botón Enviar** → Envía la consulta al asistente.
- **Tiempo de respuesta:** 10 a 20 segundos bajo condiciones normales.

---

## 3) Panel Superior Derecho

| Botón | Función |
|---|---|
| **Limpiar** | Borra el historial del chat actual. |

---

## 4) Recomendaciones de uso

- Formular preguntas claras y específicas.
- Si se habla por voz, hacerlo cerca del micrófono.
- Verificar que LM Studio esté ejecutando el modelo antes de usar el asistente.
